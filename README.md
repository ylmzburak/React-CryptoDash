# 🚀 Crypto Dash

Crypto Dash is a modern **React cryptocurrency dashboard** built with the **CoinGecko API**.  
It allows users to explore crypto coins, check live prices, and view important market data in a clean and responsive interface.

## 🌐 Live Demo

https://react-crypto-dash-beta.vercel.app/

## ✨ Features

- 📈 Live cryptocurrency prices
- 🔍 Search for crypto coins
- 💰 Market data and coin details
- 📊 Easy-to-read dashboard interface
- 📱 Responsive design for desktop and mobile
- ⚡ Fast deployment with Vercel

## 🛠️ Technologies Used

- **React**
- **JavaScript**
- **CoinGecko API**
- **Tailwind** 
- **Vercel**
